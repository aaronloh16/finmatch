# FinMatch - Swipe Right on Your Financial Future

**Built in 25 minutes at the Mini Cursor Hackathon at the Cursor Cafe event.** We were one of five groups selected to present our project live.

## What is FinMatch?

FinMatch is a Tinder-style swipe game that makes learning about investing fun and approachable. Swipe right on investments you'd consider, swipe left on ones that aren't for you, and get a personalized financial personality breakdown at the end - complete with AI-powered roasts if you pass on the wrong ones.

### Features

- **Swipe-based UI** - Familiar, intuitive card swiping powered by Framer Motion
- **AI Financial Advisor** - Claude-powered advisor that reacts to your choices with witty, personalized commentary
- **"What If" Charts** - See hypothetical growth curves for the investments you matched with
- **FOMO Toasts** - Get playfully guilt-tripped when you swipe left on solid investments
- **Rage Bait Modals** - The app fights back when you reject the sensible picks
- **Letter from Future You** - AI-generated letter from your future self based on your portfolio choices
- **Chat with Your Financial Bestie** - Post-results AI chat for personalized advice

### Investment Options Include

Boring but brilliant picks like High-Yield Savings and Index Funds, all the way to degenerate plays like Meme Coins and Your Friend's Startup (he says it's like Uber but for ironing).

## Tech Stack

- **Next.js 14** - React framework
- **Tailwind CSS** - Styling
- **Framer Motion** - Swipe animations and transitions
- **Claude API** - AI-powered advisor quips, future letters, and chat

## Getting Started

```bash
npm install
```

Create a `.env.local` file:
```
ANTHROPIC_API_KEY=your_key_here
```

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and start swiping.
